# Codex Skill: Digging

복잡한 코드 변경, 리팩터링, 문서화, 배포 전 검증을 Phase 단위로 진행하고 각 Phase마다 검증 루프를 돌리는 Codex 전역 스킬입니다.

## 설치

Codex에게 아래처럼 요청하세요.

```text
Install the skill from https://github.com/geunssam/codex-skill-digging/tree/main/digging
```

설치 후 Codex를 재시작하면 스킬 목록에 `digging`이 추가됩니다.

## 트리거 예시

```text
디깅으로 진행해.
```

```text
Phase별로 검증 루프 돌리면서 끝까지 해줘.
```
